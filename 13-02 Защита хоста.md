# Домашнее задание к занятию  «Защита хоста» - Беляева Евгения

### Задание 1

1. Установите **eCryptfs**.
2. Добавьте пользователя cryptouser.
3. Зашифруйте домашний каталог пользователя с помощью eCryptfs.


*В качестве ответа  пришлите снимки экрана домашнего каталога пользователя с исходными и зашифрованными данными.*  

### Решение 1

Создан пользователь cryptouser командой:

```
adduser cryptouser
```

В домашнем каталоге пользователя создан тестовый файл secret.txt

```
su - cryptouser
echo "secret data" > ~/secret.txt
ls -la ~
```

![host](https://github.com/evgenka5505-sketch/8-03-hw/blob/main/img/host.png)

Домашний каталог зашифрован с помощью команды

```
ecryptfs-migrate-home -u cryptouser
```
После миграции данные пользователя были перемещены в каталог: /home/.ecryptfs/cryptouser/.Private

Проверка содержимого каталога показала наличие файлов вида
```
ECRYPTFS_FNEK_ENCRYPTED.*
```
что подтверждает успешное шифрование домашнего каталога пользователя средствами eCryptfs

![host2](https://github.com/evgenka5505-sketch/8-03-hw/blob/main/img/host2.png)
