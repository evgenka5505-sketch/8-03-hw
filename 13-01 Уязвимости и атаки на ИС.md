В ходе сканирования виртуальной машины Metasploitable с помощью Nmap было обнаружено 23 открытых TCP-порта. Среди запущенных служб присутствуют FTP, SSH, Telnet, SMTP, DNS, HTTP, Samba, NFS, MySQL, PostgreSQL, VNC, IRC и Apache Tomcat.

Анализ версий сервисов показал наличие большого количества устаревшего программного обеспечения. Также был обнаружен анонимный FTP-доступ и предустановленный root shell на порту 1524.

Наиболее значимые уязвимости:

[Vsftpd 2.3.4 - Backdoor Command Execution](https://www.exploit-db.com/exploits/49757)
[Samba 3.0.20 Username Map Script Command Execution](https://www.exploit-db.com/exploits/16320)
[UnrealIRCd Backdoor Command Execution](https://www.exploit-db.com/exploits/16922)

Полученные результаты подтверждают, что Metasploitable является специально уязвимой системой, предназначенной для обучения методам поиска и эксплуатации уязвимостей.
